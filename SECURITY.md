# Security Policy

## Supported Versions

Only the latest release of SnoPeak Events is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.3.x   | :white_check_mark: |
| < 1.3   | :x:                |

## Reporting a Vulnerability

This is a private WordPress plugin for [snopeak.com](https://snopeak.com). It is not distributed publicly.

If you discover a security vulnerability, please report it by emailing **[paul@wehavevictory.com]** rather than opening a public issue.

Include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact

You can expect an acknowledgement within 48 hours. If the vulnerability is confirmed, a fix will be applied to the next release.
